Ejercicio 1 de JavaScript

Trabajando con el DOM

Tienes esta página. Como verás es muy sencilla, hemos utilizado bootstrap (última versión) para darle formato. Consta de tres columnas en dos filas, la primera con tres div con algo de texto.
La segunda fila contiene un par de botones, Modificar estilo y Añadir contenido. El primer botón lo utilizarás para invocar una función de JavaScript que cambie las clases (atributo classList en DOM) de bootstrap para cambiar los colores de la página (body), y el color de los títulos (hq y h2).
Con el botón de Añadir contenido, crearás una fila (row) debajo del row de los botones (añadir contenido para el container. Esa fila -row- tendrá cuatro columnas (div) y dentro de cada columna una lista de tres elementos (de lo que se te ocurra). Todo el JavaScript deberá estar en el archivo main.js en el mismo directorio.
